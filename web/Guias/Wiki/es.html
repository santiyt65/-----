🔩 Explicacion de Variables

Puedes Cambiar estos valores ajustando config.js y agregando el archivo a .gitignore para que el bot pueda Actualizarse sin mostrar fallas de pull.

Tambien puedes usar .setvar dato = variable pero esto reiniciara el bot en cada cambio en caso de usar Windows

⚙️ Nombre del Bot:

Se Muestran en Algunos Menus

global.wm = 'The Mystic - Bot' global.titulowm = 'The Mystic - Bot' global.titulowm2 = `The Mystic - Bot` global.igfg = 'The Mystic - Bot' global.wm2 = `${dia} ${fecha}\nThe Mystic - Bot` global.gt = 'The Mystic - Bot' global.mysticbot = 'The Mystic - Bot' global.channel = 'https://whatsapp.com/channel/0029Vaein6eInlqIsCXpDs3y' global.md = 'https://github.com/BrunoSobrino/TheMystic-Bot-MD' global.mysticbot = 'https://github.com/BrunoSobrino/TheMystic-Bot-MD'

Aqui se Agrega el número del bot y cuando inicia una nueva sesión no pedirá número sino que empezará con el código directamente. Debe ser sin espacios y sin +. No es necesario agregarlo.

global.botnumber = "595972184435"

⚙️ Administradores del Bot:

Se agrega el número, el nombre (opcional). Siempre debe tener ese Formato. Se agrega COMA "," si vas a continuar agregando más, cuando llegues al final no debe existir coma o el bot no iniciará.

global.owner = [ ['595972184435', '👑 Marco 👑', true], ['5218444937407'] ]
